# nlp-social-media-analytics
# NLP Powered Social Media Analytics Framework

This project performs:

- Sentiment Analysis
- Topic Modeling
- Engagement Prediction
- Social Media Recommendation

Technologies Used:
- Python
- NLP
- Scikit-learn
- Streamlit
- TF-IDF
- Logistic Regression
- LDA

## Run Project

```bash
streamlit run app.py
```
